# Issue Tracker

- **Provider** : GitHub
- **Dépôt** : `gaspardViel/i_have_an_idea`
- **CLI** : `gh` (GitHub CLI)

## Commandes courantes

```bash
gh issue create --title "..." --body "..."   # créer une issue
gh issue list                                 # lister les issues ouvertes
gh issue view <numéro>                        # lire une issue
gh issue close <numéro>                       # fermer une issue
gh issue edit <numéro> --add-label "..."      # ajouter un label
```
