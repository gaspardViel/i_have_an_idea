# Mission d'apprentissage

## Qui

Gaspard. Écrit une autobiographie/mémoire. Brouillon substantiel en cours.

## Pourquoi apprendre Claude Code / agents IA

Accélérer l'écriture du livre. Aller au-delà du copier-coller avec ChatGPT/Claude web.

## Point de départ

- Maîtrise les LLMs via interface web (ChatGPT, Claude.ai)
- Pas d'expérience avec la CLI, les agents, ou VS Code + IA
- Livre : mémoire autobiographique, brouillon avancé

## Objectif final

Utiliser Claude Code (ou un agent équivalent) directement dans VS Code pour :
- Travailler **dans** les fichiers du manuscrit (pas du copier-coller)
- Maintenir la cohérence du livre (voix, chronologie, personnages)
- Accélérer la rédaction des chapitres restants
- Réviser et éditer avec un assistant qui connaît tout le contexte du livre

## Ce que le succès ressemble

Gaspard ouvre VS Code, dit à Claude "révise le chapitre 7 pour qu'il aille mieux avec le ton du chapitre 2" — et Claude le fait directement dans le fichier, sans copier-coller.
