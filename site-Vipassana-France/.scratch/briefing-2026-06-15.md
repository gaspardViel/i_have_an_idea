# Briefing projet — Vipassana France — 15 juin 2026

## En une phrase

Migration du Site Vipassana France de Typo3 vers WordPress, avec un Template moderne à vocation européenne, maintenable par des Bénévoles techniques non-développeurs.

---

## Contexte et vision

**Problème :** Le Site actuel (fr.dhamma.org) tourne sur Typo3, trop complexe pour les bénévoles. Il est sous-maintenu et visuellement daté.

**Solution :** Un WordPress unique pour tout le réseau France, simple à éditer, avec un Template sobre et documenté — conçu pour être proposé au réseau Dhamma Europe.

**Ambition long terme :**
1. Site Vipassana France exemplaire sur WordPress
2. Template proposé comme référence au réseau Dhamma Europe
3. Workflow de maintenance bénévole + agent IA (futur)

**État actuel :** WordPress déjà installé avec un thème et du contenu partiel, mais insatisfaisant. Le travail commence par un audit de ce qui existe.

**Équipe :** Gaspard (pilote, setup technique) + Éric (co-développeur)

---

## Décisions prises

| Décision | Choix |
|----------|-------|
| Migration de contenu | Tout migrer — aucune perte (photos, textes, liens) |
| Architecture | Un seul WordPress (4 Centres + France générale) |
| Éditeur | Gutenberg natif — pas d'Elementor |
| Multilingue | Polylang FR (complet au lancement) + EN (progressif) |
| Section Anciens étudiants | Mot de passe partagé natif WordPress |
| Agent IA | Mode "propose, humain valide" — pas d'écriture autonome |

## Vocabulaire clé

> Utiliser ces termes partout — dans les issues, les discussions, le code.

| Terme | Définition | À éviter |
|-------|-----------|----------|
| **Cours** | Retraite de méditation structurée dans un Centre (typiquement 10 jours) | Retraite, Stage, Session |
| **Centre** | Lieu physique où se déroulent les Cours | Site (ambigu) |
| **Site** | Le site web Vipassana France | Page |
| **Nouvel étudiant** | Personne n'ayant pas encore fait de Cours de 10 jours | Débutant, Yogi |
| **Ancien étudiant** | A complété au moins un Cours de 10 jours — accès Section protégée | Méditant, Yogi |
| **Servant du Dhamma** | Ancien étudiant qui contribue bénévolement à un Centre | Serviteur, Bénévole |
| **Calendrier** | Planning des Cours — hébergé sur dhamma.org, jamais sur le Site | Agenda |
| **Inscription** | S'enregistrer pour un Cours — toujours via dhamma.org | Réservation |
| **Section Anciens étudiants** | Partie du Site protégée par mot de passe partagé | Espace membres |
| **Template** | Thème WordPress cible, à proposer au réseau Dhamma Europe | Thème, Design |
